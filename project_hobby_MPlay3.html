MPlay3 - Music Player





After I got my first and only Apple product, the 2nd generation iPod, I started using iTunes as a music player. For quite some time I was happy with it and especially liked the feature that you could have columns for genre, artist, album and songs next to each other. If you wanted to listen to specific genres you could click on those in the corresponding column and the other columns would adjust and display the correct artists, albums and songs that belong to the chosen genres. This obviously worked for the other columns as well. But at some point this feature was destroyed in an update, which prompted me to never update my iTunes again. But as I upgraded to Windows 10 I stopped trying to keep the old version of iTunes, as it was too much of a hassle. From then on I used VLC to play my music, yes I know, but I was annoyed trying to find a replacement and gave up.
Then I got the idea to just create my own music player, which seemed feasable as I only have .mp3 files for my music. To make it more interesting I tried to do everything myself. The only libraries that are used in this project is the STB_Truetype library and the MiniMP3 library which are both single header-files, but those should be deprecated in the long term as well. The whole prototype is written in, what I call, C+. C+ is C99 with very limited use of C++ features like operator overloading.

MPlay3 is now also on the handmade network, at this location. On there I give a few more details and also the source code linked, if anyone is interested.
